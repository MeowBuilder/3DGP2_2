# 프로젝트 요약 (Project Summary)

## 현재 구조 (Current Structure)
- `MainMenu`와 `InGame` 두 가지 상태를 관리하는 게임 상태 머신(State Machine)이 도입되었습니다.
- 프로그램 시작 시 `MainMenu` 상태가 활성화되어, `SkyBox/SkyBox_0.dds` 텍스처를 사용한 시작 화면이 표시됩니다.
- 사용자가 Enter 키를 누르면 `InGame` 상태로 전환되어 3D 게임 월드가 렌더링됩니다.
- UI 렌더링을 위해 2D 텍스처 전용 `UIShader`와 화면을 덮는 사각형 메쉬 `ScreenQuadMesh`가 사용됩니다.
- '참고용' 폴더의 코드가 프로젝트에 성공적으로 통합되었습니다.

## 다음 계획 (Next Steps)
- 시작 화면 기능 구현이 완료되었습니다.
- 다음 작업 지시를 대기 중입니다.
